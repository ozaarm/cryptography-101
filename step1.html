Cryptography for Beginners

1. Shifting characters

Cryptography is the science of writing secret messages.

It is widely used in telecommunications, mobile apps (Whatsapp) and generally enables safe sharing of sensitive information such as payment details. It's one of the cornerstone of blockchain technologies.

Cryptography and encryption are often confused, but they are not the same. Cryptography is the broader field that studies how to protect information through mathematical techniques, including authentication, hashing, digital signatures, and more.

Encryption is one technique within cryptography that focuses specifically on scrambling a message so only the intended recipient can read it. Encoding is the process of converting data into a different format so it can be safely stored or transmitted — not to hide its meaning, but to ensure compatibility.

One of the reasons for the confusion is because all those techniques use keys. Let's find out how this works with super practical examples.

How It Works

Every character (a, b, c, 1, 2, 3, !, =, ?) has a hidden numeric code. The programme extract the first character of the input (0 = first one), gets its numeric code with .charCodeAt(0), then increments it by 1 with String.fromCharCode(code + 1) and finally displays the result in the output box.

const input = document.getElementById('charInput').value;
const code = input.charCodeAt(0);
const shifted = String.fromCharCode(code + 1);

⚠️ Any character can be transformed into something else by changing its numeric value.

How It Works

A for loop scans each character of the input string, gets the code with charCodeAt(i) for each, then increments the code by 1 and converts it back with fromCharCode(). Spaces and punctuation are also shifted because they also have character codes.

let output = "";
for (let i = 0; i < input.length; i++) {
  let code = input.charCodeAt(i);
  output += String.fromCharCode(code + 1);
}

and then loop back each character with decrypted += String.fromCharCode(code - 1);

Take-Aways

Shifting one character by 1 in words (and sentences) is quite easy, but the pattern is quite recognizable. For instance we quickly notice that spaces are always translated as exclamation points (!). We need to explore something more elaborate.
