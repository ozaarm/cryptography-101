Getting Started

Cryptography is used everywhere from WhatsApp to Signal to ... crypto currencies.

But how does it work actually? What are those keys I keep hearing about ? I decided to find out with a super practical tutorial for beginners (like me).

Example

First, let's start with a super basic form of cryptography where we replace a letter by the following one : a = b, b = c, c = d, etc.



OUTPUT

How It Works

Every letter (a, b, c) has a numeric code. We take the first character of the input value (character 0 is the first), we get its numeric code with .charCodeAt(0), increment it by 1 with String.fromCharCode(code + 1), we store it before displaying it in the output box.

const input = document.getElementById('charInput').value;
const code = input.charCodeAt(0);
const shifted = String.fromCharCode(code + 1);
